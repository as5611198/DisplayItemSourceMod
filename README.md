# DisplayItemSourceMod
DisplayItemSourceMod (範例)

這是一個範例 Mod，展示如何使用 DuckovCoreAPI。

功能

在滑鼠移到物品上時，顯示該物品的來源 Mod 或「遊戲本體」。

依賴 

[必要] DuckovCoreAPI: 你必須先安裝核心 API。

Steam 工作坊連結 (推薦) https://steamcommunity.com/sharedfiles/filedetails/?id=3603671043

GitHub Releases 連結 https://github.com/as5611198/DuckovCoreAPI/tree/ff7adf30876bfbe12883a7f021a147f98311984f

如何安裝 (給玩家)

必須先安裝 DuckovCoreAPI (見上方的連結)。

方法一 (推薦)：Steam 工作坊

前往 Steam 工作坊頁面：[[請點此處訂閱]](https://steamcommunity.com/sharedfiles/filedetails/?id=3603671043) (請填入你自己的連結)

點擊「訂閱」。
